## Paso 1: Descargar y preparar herramientas

En este caso se utilizó **Blender** como herramienta principal de modelado.
### Instalación de Blender en Windows

1. **Descargar Blender:**  
    Visita la página oficial [https://www.blender.org/download/](https://www.blender.org/download/) y selecciona la versión más reciente para Windows.
    
2. **Ejecutar el instalador:**  
    Una vez descargado el archivo `.exe`, ábrelo y sigue las instrucciones del asistente de instalación.
    
3. **Elegir carpeta de destino:**  
    Puedes dejar la ruta por defecto (por ejemplo: `C:\Program Files\Blender Foundation\Blender`) o definir una personalizada.
    
4. **Finalizar e iniciar:**  
    Al finalizar la instalación, marca la opción **“Run Blender”** para abrir el programa por primera vez.
    
5. **Configuración inicial recomendada:**
    
    - Cambia el idioma si lo prefieres: _Edit > Preferences > Interface > Translation > Language: Español (es)_.
        
    - Ajusta la escala de la interfaz según tu pantalla.
        
    - Guarda tus preferencias haciendo clic en _Save Preferences_.
        
6. **Descarga e instala los complementos recomendados (Add-ons):**
    
    - **Loop Tools:** activarlo desde _Edit > Preferences > Add-ons > Search: “Loop Tools” > Activar_.  
        Este add-on facilita operaciones como Bridge, Circle, Relax y Flatten durante el modelado.
        

Con esto, Blender quedará listo para comenzar el modelado del personaje.

## Paso 2: Modelado del cuerpo base

### 2.1 Tronco

1. **Eliminamos la luz y la cámara:**  
    Selecciónalas en el _Outliner_ y presiona **X → Delete**.
    
2. **Iniciamos desde un cubo básico:**
    
    - Añade un cubo con **Shift + A → Mesh → Cube**.
        
    - Cambia a **Vista Frontal (Numpad 1)**.
        
    - Activa el modo **Ortográfico (Numpad 5)** para evitar deformaciones visuales.
        
3. **Colocación del cubo según la referencia:**  
    El centro del cubo debe quedar a la altura del ombligo de la referencia humana.  
    Ajusta la parte superior e inferior escalando (**S**) y moviendo (**G**).
    
    ![[Pasted image 20251104133011.png]]
    
4. **Ajuste según la vista frontal y lateral:**
    
    - Trabaja en la **vista frontal (Numpad 1)** para igualar la silueta.
        
    - Alterna constantemente con la **vista lateral (Numpad 3)** para mantener volumen.
        
    - Usa **Alt + Z** para activar el modo de transparencia (_X-Ray_) y ver los vértices traseros.
        
    
    ![[Pasted image 20251104133223.png]]  
    ![[Pasted image 20251104133346.png]]
    
5. **Añadir cortes (Loop Cuts):**
    
    - Desde la vista frontal, agrega **3 bucles verticales** con **Ctrl + R**.
        
    - Desde la vista lateral, agrega **2 cortes horizontales** con el mismo atajo.
        
    
    ![[Pasted image 20251104133934.png]]
    
6. **Suavizado de vértices (Smooth Vertices):**  
    Selecciona los vértices del cubo original y aplica:
    
    - _Vertex > Smooth Vertices_ o presiona **Ctrl + V → Smooth Vertices**.
        
    - Usa un valor de **Smoothing: 0.792** para redondear la forma base.
        
    
    ![[Pasted image 20251104134659.png]]
    
    Los demás vértices también deben ser suavizados manualmente para lograr una silueta fluida.
    
7. **Aplicar el modificador Mirror:**
    
    - Elimina la mitad de las caras (en modo edición, selecciona la mitad y presiona **X → Faces**).
        
    - Luego añade el **modificador Mirror** desde el panel de propiedades (icono de llave inglesa).
        
    - Asegúrate de que la opción **Clipping** esté activada para evitar separación en el eje central.
        
    
    ![[Pasted image 20251104135205.png]]

### 2.2 Piernas

1. **Agregar el corte vertical para las piernas:**
    
    - En modo edición (**Tab**), usa **Ctrl + R** para hacer un corte justo en el centro inferior del cuerpo.
        
    - Ajusta los vértices para separar la silueta de las piernas según la referencia.  
        ![[Pasted image 20251104140155.png]]
        
2. **Eliminar caras para extruir las piernas:**
    
    - Selecciona y elimina **6 caras** en la parte inferior del cuerpo (**X → Faces**).  
        ![[Pasted image 20251104140440.png]]
        
3. **Antes de extruir:**
    
    - Crea **más cortes verticales** (**Ctrl + R**) y suavízalos con **LoopTools → Relax** o con _Smooth Vertices_ (**Ctrl + V → Smooth Vertices**).  
        ![[Pasted image 20251104140819.png]] ![[Pasted image 20251104140924.png]]
        
    - Esto nos permitirá tener **12 vértices por bucle** en cada pierna.  
        ![[Pasted image 20251104141050.png]]
        
4. **Acomodar la geometría inferior del cubo:**
    
    - Ajusta los vértices inferiores para que las piernas nazcan correctamente.
        
    - Recuerda suavizar todos los cortes con _Smooth Vertices_.  
        ![[Pasted image 20251104141438.png]] ![[Pasted image 20251104141621.png]]
        
5. **Extruir las piernas:**
    
    - Selecciona el _loop_ de aristas inferior y extruye (**E**) hasta los tobillos.
        
    - Igualar la altura de los vértices en el eje Z con **S + Z + 0**.
        
    - Escala para que coincida con el tobillo de la referencia (**S**).  
        ![[Pasted image 20251104142014.png]]
        
6. **Redondear la base de las piernas:**
    
    - Utiliza **LoopTools → Circle** para hacer el bucle perfectamente circular.  
        ![[Pasted image 20251104142217.png]]
        
    - Rota el bucle (**R**) para que las caras verticales queden alineadas con la rodilla.  
        ![[Pasted image 20251104142315.png]]
        
7. **Definir la forma general:**
    
    - Agrega más cortes (**Ctrl + R**) en las piernas para definir su forma.
        
    - Alinea los bucles con **S + Z + 0** para mantenerlos nivelados.  
        ![[Pasted image 20251104142933.png]]
        
8. **Usar la herramienta “Boolean Tool” (Atajo: B + mover el mouse)** para añadir nuevos bucles rápidamente.
    
    - Rueda el mouse para ajustar la cantidad de cortes.  
        ![[Pasted image 20251104143041.png]]
        
9. **Formar la rodilla:**
    
    - Ajusta los vértices lateralmente para darle volumen a la rodilla.  
        ![[Pasted image 20251104143246.png]]
        
10. **Acomodar tobillo y definir detalle:**
    
    - Asegúrate de alinear la zona del tobillo con la referencia.
        
    - Crea **4 cortes adicionales** sobre las rodillas (**Ctrl + R**) para definirlas mejor, y alinea con **S + Z + 0**.  
        ![[Pasted image 20251104143512.png]]
        
11. **Definir espinillas y pantorrillas:**
    
    - Añade 4 cortes más en las piernas para formar el volumen de los músculos.
        
    - Ajusta cada sección con cuidado, usando las vistas **frontal (Numpad 1)**, **lateral (Numpad 3)** y **perspectiva (Numpad 5)**.  
        ![[Pasted image 20251104143739.png]]
        
    - No olvides revisar las **partes laterales** para mantener una forma realista.  
        ![[Pasted image 20251104143935.png]]

    - Añade **3 bucles** como en las rodillas (**Ctrl + R**) para definir la forma.
